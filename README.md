# blog6 -- A Blog System for Markdown

## Install

```
$ npm i blog6 --global
```

## Run

```
$ blog6 --root=<path-to-blog-root> --port=<port>
```

Example

```
$ blog6 --root=./ --port=80
Blog6 run at http://localhost:80 with root ./
```

## Blog Host -- Sample


```
$ git clone https://gitlab.com/ccckmit/course.wiki.git
$ blog6 --root=./course.wiki --port=80
```
